# Publish Web Apps Compatibility Router

This name is retained only so older prompts do not activate obsolete runtime
publishing guidance.

Read and follow the sibling `finite-sites-publishing-finite` skill. Current
publishing uses the `fsite` Project Repository and Project Output model for
static websites, Markdown documents, and stateful apps.

Do not expose a Runtime port, edit host networking, or substitute another
Finite command family. If `fsite` is unavailable or too old for the requested
output type, stop and report the missing product capability.
